   <파이선 프로젝트 요구사항>
   
   1. Data 분석용 파이썬 라이브러리를 구축
   
   - csv flie을 읽어서 data frame을 작성한다. 
   
   - data frame 내용을 출력한다.
   - data frame의 변수 속성을 출력한다.
   - data frame의 data와 변수의 수를 출력한다. 
   - data frame의 변수 항목을 리스트로 만든다.
   - data frame의 앞부분 일정 부분의 내용을 출력한다.
   - data frame의 뒷부분 일정 부분의 내용을 출력한다.
   
   - data frame의 변수 이름을 변경하는 기능을 제공한다.
   - data frame에 파생 변수를 추가하는 기능을 제공한다.
   - data frame에서 필요한 변수를 추출하는 기능을 제공한다.
   - data frame에서 조건에 따른 data를 추출하는 기능을 제공한다.
   - data frame의 data를 정렬하는 기능을 제공한다.( 오름/내림차순 기능 별도 제공 )
   
   - 데이터 요약 기능을 제공한다.( 다음 값들을 일괄 계산하여 모두 출력 )
         * 변수의 최소값
	 * 변수의 최대값
	 * 변수의 합계
	 * 변수의 평균
   - 빈도 분석 기능을 제공한다.
   - 합계를 계산하는 기능을 제공한다.
   - 그룹별 합계를 계산하는 기능을 제공한다.
   
   - data 결합 기능을 제공한다.
   - 변수 결합 기능을 제공한다.
   
   - 결측치 처리 기능을 제공한다.( 확인, 제거, 대체 )
   - 이상치 처리 기능을 제공한다.( 확인, 제거, 대체 )
   
   - 막대 그래프 출력 기능을 제공한다.
   - 텍스트 마이닝 기능을 제공한다(옵션)
 
   * 위 기능을 가급적 모두 구현하고, 필요하다면 기능을 추가하는 것도
     가능
     
2. 제출 결과물
   2.1 작성한 함수/클래스 저장 .py 파일
   2.2 작성한 함수/클래스에는 반드시 작성자 이름, 기능 설명을 
       python 여러줄 문자열로 함수명/클래스명 다음줄에 기술한다.
   2.3 작성한 라이브러리를 이용하여 데이터 분석에 사용한
       csv 파일
   2.4 작성한 라이브러리를 이용하여 데이터 분석한 분석 결과 보고서

3. 발표
   3.1 작성한 라이브러리 전체 구조 설명
   3.2 작성한 라이브러리 각각의 실행 동작 시연
   3.3 작성한 라이브러리를 이용한 데이터 분석 과정 시연
   3.4 데이터 분석 결과 발표
